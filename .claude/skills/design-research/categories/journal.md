# Journal vertical scaffold

For mood / gratitude / habit / diary / reflection apps.

## Reference apps

1. **Daylio** — gold standard for mood tracking, mood-color-emoji home, calendar view.
2. **Stoic** — gratitude + reflection prompts, calm aesthetic, AM/PM split.
3. **Reflectly / Jour** — AI-prompted journaling, conversational entry flow.
4. **Streaks** — habit tracking, ring-per-habit pattern.
5. **Notion / Day One** — long-form journal benchmark, photos and rich entries.
6. **Finch** — gamified self-care, character pet pattern (cautionary if your brand isn't playful).
7. **How We Feel** — emotion-wheel input, research-backed, beautiful typography.

## Vertical questions

### Entry flow
- Quick entry (mood + 1-line) vs long-form?
- Voice-to-text input?
- Photos / location / weather attached automatically?
- Time-of-day prompts: morning intention + evening reflection?
- Streak-aware: does the entry CTA change if user is about to break streak?

### Mood / state input
- Scale 1–5 emoji vs color wheel vs slider vs custom emotion list?
- "Why?" follow-up: optional tag list (work, family, sleep, exercise...)?
- Free-text required or optional?

### Calendar / history view
- Month grid colored by mood?
- Streak visible at top?
- Tap a day to see entry?
- Search across entries?

### Insights / patterns
- "You feel better on days you exercise" — correlation surfaces.
- Mood trend line over weeks.
- Most common tags / triggers.
- Word cloud of journal text (privacy-preserving, all local).

### Privacy
- Lock with FaceID / passcode (table stakes for journal apps)?
- iCloud / sync: opt-in or required?
- Export: PDF, encrypted backup?

## Customization knobs

1. Reminder time (morning / evening / both)
2. Mood scale type (5-point / 7-point / wheel)
3. Custom tags / categories
4. Privacy lock toggle
5. Weekly summary email/notif

## Anti-patterns

- ❌ Forcing long entries — most users want 30s in/out.
- ❌ Notification spam ("you haven't journaled in 2 days" multiple times).
- ❌ Social / sharing as primary feature (this is a private category).
- ❌ Gamification that feels childish on a serious-mood app.
- ❌ Bright saturated palette — most users want calm.

## Visual signals

- Soft, warm palettes (sand, dusty pink, sage).
- Serif typography for entries (Lora, Crimson, Merriweather) to feel "writing", not "form".
- Photography-led when app is photo-rich (Day One).
- Dark mode is essential — many people journal in bed.
