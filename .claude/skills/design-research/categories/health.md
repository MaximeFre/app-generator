# Health vertical scaffold

For sleep / meditation / hydration / cycle / breathing / general wellness apps.

## Reference apps

1. **Headspace** — gold standard for meditation, character-led illustrations, calming pace.
2. **Calm** — sleep stories, atmospheric photography, scrubber-based audio.
3. **Oura / Whoop** — sleep + recovery, ring/donut metric pattern.
4. **Flo / Clue** — cycle tracking, calendar central, prediction language.
5. **AutoSleep** — Apple Watch sleep, beautiful sleep ring + bedtime/waketime visual.
6. **WaterMinder** — hydration, glass-fill animation, simple log.
7. **Apollo Neuro / Othership** — breathwork, full-screen guided session.

## Vertical questions

### Daily home / today
- One number to rule them all (recovery, sleep score, readiness)?
- "Today's recommendation" generated based on yesterday's data?
- Streak of consecutive days?

### Logging entry
- Auto from Apple Health / Google Fit vs manual?
- Quick log buttons (drank water, took meditation) on home?
- Time-of-day tags (morning, evening) auto-applied?

### Guided sessions (meditation / breathwork)
- Library tab vs single "play" focus?
- Categories: by goal (sleep, stress, focus) vs by length?
- Personalized recommendation ("for you")?
- Streak / minutes-meditated tracking?

### Visualizations
- Sleep stages bar chart (REM / Deep / Light / Awake)
- Cycle calendar with predicted phases
- Hydration glass-fill ring
- Streak heatmap

### Privacy (very high stakes)
- Health data is HIPAA-adjacent.
- Don't sync to cloud without explicit consent.
- Don't share with social features by default.

## Customization knobs

1. Bedtime / wake time goal
2. Daily water goal (oz / ml)
3. Reminder cadence (every N hours)
4. Cycle length (28 days default, customizable)
5. Notification quiet hours

## Anti-patterns

- ❌ Cluttered home with 6 widgets — pick the one number that matters today
- ❌ Bright saturated palettes — health apps need calm
- ❌ Streak shame ("you broke your streak!") — soft language only
- ❌ Comparison with others by default (privacy concern)

## Visual signals

- Soft palettes — sage, dusty blue, lavender, muted coral.
- Lots of whitespace.
- Photography-led for sleep / meditation; illustration-led for cycle / hydration.
- Audio-first when sessions are involved (big play button, scrubber).
- Animations are slow and breathing-paced, never snappy.
