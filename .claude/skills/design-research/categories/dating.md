# Dating vertical scaffold

For match / message / profile apps. **Warning**: dating apps depend heavily on liquidity (real users on both sides). Indie viability requires niche focus.

## Reference apps

1. **Hinge** — best UX pattern for indie inspiration: profile-as-conversation-prompts, like-on-specific-prompt.
2. **Tinder** — swipe baseline, photo-stack pattern.
3. **Bumble** — women-message-first rule, time-limited conversations.
4. **Feeld / Lex** — niche communities, identity-rich profiles.
5. **The League / Raya** — selective + status patterns.
6. **Coffee Meets Bagel** — daily curated batch.

## Vertical questions

### Profile creation
- How many photos required vs optional?
- Prompts vs free-form bio?
- Verification (selfie match)?
- What identity fields: age, location, gender, orientation, height, drinking, smoking, kids, religion?

### Match flow
- Swipe-stack (Tinder) vs like-specific-prompt (Hinge)?
- Like + comment vs like-only?
- Limit per day (creates scarcity)?
- See who liked you (premium gate)?

### Match → message
- Auto-match notification or 24h-to-message pattern?
- Conversation starter prompts?
- Read receipts (off by default)?

### Discovery
- Distance filter slider
- Age range
- Looking for (relationship / casual / friends)
- "Boost" / "Spotlight" (premium)

### Safety
- Block / report from any surface (table stakes)
- "Are you safe?" check-in feature (Tinder)
- Photo verification badge

## Customization knobs

1. Distance radius
2. Age range
3. Visibility (active / hidden)
4. Read receipts
5. Quiet hours for notifications

## Anti-patterns

- ❌ Empty app at launch — seed market or pick a tiny geo first
- ❌ Showing sub count on dashboard ("Premium! Unlock matches!") — too aggressive
- ❌ Photo-only profiles for serious relationship apps — Hinge wins by being prompts-first
- ❌ Public stats (you have 12 likes) — pressure-y
- ❌ Forcing location at sign-up before showing any value

## Visual signals

- Photography-led, large images.
- Soft, romantic palettes (Hinge: dark blue + cream) OR vibrant (Tinder: orange + pink).
- Card-stack with rotation on swipe.
- Bottom action buttons (heart, X, comment) prominent and large.
- Profile detail uses strong vertical hierarchy: photo, prompt, photo, prompt.
