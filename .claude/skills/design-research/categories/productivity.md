# Productivity vertical scaffold

For tasks / notes / time-tracking / GTD / focus apps.

## Reference apps

1. **Things 3** — gold standard for IA, "Today" + "Anytime" + "Someday" buckets, magic + button.
2. **Todoist** — natural language input ("buy milk tomorrow at 5pm"), labels system.
3. **Bear / Craft / Linear Notes** — markdown, beautiful editor patterns.
4. **Notion mobile** — block-based, capture-first.
5. **Centred / Session** — focus timer + today's task pattern.
6. **TickTick** — habit + task hybrid.
7. **Toggl / Timery** — time tracker, big timer button + project list.

## Vertical questions

### Capture flow
- Quick add via FAB always visible?
- Natural-language parsing for date/time?
- Voice input?
- Inbox vs direct project assignment?

### Today / overview
- "Today" as a virtual list (filter), not a separate database
- Overdue grouping vs hidden?
- Drag to reorder priority?
- Keyboard shortcuts (rare on mobile but power users like)?

### Project / list management
- Grouping: by project / area / tag?
- Nested projects?
- Templates?

### Focus / Pomodoro
- Timer mode separate from tasks or integrated?
- Session log → time-tracking link?
- Distraction blocker (iOS Focus integration)?

## Customization knobs

1. Default reminder time
2. Week start day
3. Smart list (Today, This Week, Next 7 days) toggle per list
4. Notification per list / project (on/off, time)
5. Default project for inbox

## Anti-patterns

- ❌ Settings buried; users want quick access to "edit reminder time"
- ❌ Complex initial onboarding asking for projects, areas, tags BEFORE first task
- ❌ Heavy gamification (productivity apps must feel professional)

## Visual signals

- Either ultra-minimal (Things, Bear) OR data-dense (Linear) — pick one, don't half-commit.
- Sans-serif system or refined neo-grotesque (Inter, Geist).
- Single accent color, often blue or graphite. Heavy use of muted gray for chrome.
- Strong typographic hierarchy (16/14/12px).
